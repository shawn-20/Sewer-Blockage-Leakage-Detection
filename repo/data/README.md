## Data

### BattleDIM Dataset (Phase 1)
This project uses the BattleDIM L-Town benchmark dataset.

- **Source:** KIOS Research Center, University of Cyprus
- **DOI:** https://doi.org/10.5281/zenodo.4017659
- **GitHub:** https://github.com/KIOS-Research/BattLeDIM
- **Licence:** EUPL v1.1

Download the dataset from Zenodo and place the CSV files 
in this folder before running the notebooks.

Expected files:
- 2018_SCADA_Pressures.csv
- 2018_SCADA_Flows.csv
- 2018_AMR.csv
- 2018_Leakages.csv
- (+ any additional CSVs from the v1-v4 dataset files)
